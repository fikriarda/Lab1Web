Paragraf merupakan kumpulan dari beberapa kalimat. Pada dokumen web, paragraf biasanya digunakan untuk menampilkan teks atau artikel.
Paragraf pada dokumen HTML dibuat dengan tag.

Ada beberapa tag yang dapat digunakan untuk memformat teks pada Paragraf.
Diantaranya adalah Bold Italic Importan text Emphasized text Marked text Small Text Deleted Text Inserted Text Subcript Text Superscript Text
